## Install
1. Clone this repo
2. Install dependencies
   ```sh
   # yarn is fine too
   npm i
   ```
3. Run bootstrap
   ```sh
   npm run bootstrap
   ```

   This will bootstrap and link between app and lib
4. Now, it's ready!

## Dev flow
1. Run start
   ```sh
   npm run start:app
   ```

   If everything is alright, you will see:
